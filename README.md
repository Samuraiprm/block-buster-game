# Блок Бастер - Головоломка 🧩

Интерактивная головоломка в стиле Block Buster с бесконечными подсказками и случайным расположением блоков.

## 🎮 Как играть

1. **Нажимайте на блоки** одного цвета, чтобы их уничтожить
2. **Совпадения 3+ блоков** одного цвета исчезают
3. **Блоки падают** вниз и заполняют пустые места
4. **Цель** - набрать как можно больше очков за минимальное количество ходов

## ✨ Особенности

- **Случайная генерация уровней** - каждый уровень уникален
- **Бесконечные подсказки** - нажимайте "💡 Подсказка" столько, сколько нужно
- **Нет рекламы** - играйте без отвлекающих факторов
- **Красочный дизайн** - яркие градиенты и анимации
- **Адаптивный интерфейс** - работает на компьютерах и телефонах

## 🎯 Управление

| Действие | Как сделать |
|----------|-------------|
| Выбрать блок | Нажмите на него |
| Убрать выбор | Нажмите на тот же блок |
| Подсказка | Нажмите кнопку "💡 Подсказка" |
| Новый уровень | Нажмите кнопку "🔄 Новый уровень" |
| Начать сначала | Нажмите кнопку "◀️ Начать сначала" |

## 📊 Система очков

- Каждый уничтоженный блок дает очки равные его значению × 10
- Чем больше блоков уничтожите за один ход, тем больше очков
- Значения блоков отображаются на них самих
